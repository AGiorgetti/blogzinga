## BlogZinga! ##

Find your favorite blogs here!

[http://www.blogzinga.it](http://www.blogzinga.it)

### How to build ###
* install [node.js](http://nodejs.org/)
* in *root* folder
```
npm install && bower install
```
* in *backend* folder
```
npm install
```
### How to lauch ###
* in *root* folder
```
gulp
```

Check your browser at
  http://localhost:8000/
